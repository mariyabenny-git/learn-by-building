Student Portfolio
A personal profile page that loads all content from a JSON file.
Folder Structure
📁 student-portfolio/
  ├── index.html
  ├── style.css
  ├── script.js
  └── profile.json
Technical Schema (profile.json)
{ "name": "Your Name", "role": "Developer / Designer", "skills": ["HTML", "CSS", "JavaScript"], "projects": [ { "title": "Project Name", "description": "Brief details..." } ] }
Task Checklist
  1. 1 Set up the 4 files — index.html, style.css, script.js, profile.json
  2. 2 Build the HTML skeleton using semantic tags
  3. 3 Fill profile.json with your own profile data
  4. 4 Use fetch() in script.js to render your UI sections
  5. 5 Style layout with Flexbox and CSS Grid properties
  6. 6 Implement clean layout responsive media queries
  7. 7 Add client interactive smooth scroll transitions
  8. 8 Deploy project repository on GitHub Pages hosting
Bonus Challenges
  • Add a Dark/Light theme toggle switch powered by LocalStorage state.
  • Incorporate tiny hover animations on skill micro-badges.

  6. 6 Connect structural global query search input filters
  7. 7 Render comprehensive target step details modal layers
  8. 8 Standardize fully adapted desktop and viewport views
Bonus Challenges

Quiz App
An interactive trivia game mapping response validations from JSON records.
Folder Structure
📁 quiz-app/
  ├── index.html
  ├── style.css
  ├── script.js
  └── questions.json
Technical Schema (questions.json)
[ { "question": "What does CSS stand for?", "options": [ "Creative Style Sheets", "Cascading Style Sheets", "Computer Style Sheets" ], "correct": 1 // Index matching correct array choice } ]
Task Checklist
  1. 1 Build initial boilerplate document mapping definitions
  2. 2 Setup comprehensive objective problem arrays layout
  3. 3 Design initial setup screens alongside display cards
  4. 4 Stream single collection item dynamically per iteration
  5. 5 Validate user select matching index configurations
  6. 6 Keep session score totals and progression status bars
  7. 7 Display performance percentage aggregates screens metrics
  8. 8 Polish feedback color triggers and fade transitions
Bonus Challenges
  • Incorporate a responsive 15-second visual countdown timer per problem card.
  • Generate a final breakdown log reviewing incorrect selections at summary completion.
General Tips
  • Use browser DevTools (F12 or right-click → Inspect) to monitor Console warnings and troubleshoot dynamic script fetch issues.
  • Validate that your JSON file structure matches the technical schemas exactly (ensure commas and quote formats are precise).
  • Commit your workflow changes into your GitHub repository milestones cleanly before firing your production build pages validation logic.
